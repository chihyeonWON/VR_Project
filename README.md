# Unity_Project
Unity_Project on 2023 Summer Vacation.

## Project Idea
```
자주 가는 헬스장의 기구나 인테리어를 Unity 3D의 에셋들로 옮겨서
1인칭 헬린이가 운동을 통해 레벨업(근성장)을 하는 과정을 담은 1인칭 Unity Game 프로젝트

인바디의 각종 근성장 지표 수치를 게임의 레벨업 시스템의 수치로 변환
```
